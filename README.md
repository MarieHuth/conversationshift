# Conversationshift
### AI-Powered Manager Copilot · A Proof of Concept by Marie Huth

## What is this?
Conversationshift is an agentic learning tool that helps managers prepare for difficult conversations — built to demonstrate what embedded, in-the-flow-of-work learning looks like in practice.

Rather than a course or a workshop, Conversationshift meets managers where they already are: in the moment, with a real situation, needing real guidance right now.

## The DATA Framework
Conversationshift is built around DATA — a framework designed for analytical, results-oriented managers who need structure without sentimentality:

- **D** — Describe the situation
- **A** — Articulate the behavior  
- **T** — Talk through impact
- **A** — Agree on action

## How it works
1. Manager selects their conversation type
2. Conversationshift asks targeted clarifying questions one at a time
3. Personalized DATA talking points are generated
4. Manager iterates — refining language, stress-testing pushback, sharpening action steps

## Why I built this
This POC demonstrates my approach to AI enablement in talent development: move beyond event-based learning toward agentic, continuous, embedded experiences that generate value in the moment of need.

Built with React and the Anthropic Claude API.
